# Koddi Reservation Campaign Builder Skill

Automates creation of a Koddi reservation campaign (reservation + multiple ad groups + submit verification).

## Run

```bash
./skills/koddi-reservation-campaign-builder/scripts/run-koddi-campaign.sh /absolute/path/to/campaign.json
```

## Required Input

Use the template at:

- `skills/koddi-reservation-campaign-builder/references/campaign.template.json`

## NPM Script

```bash
npm run create:koddi-reservation
```
